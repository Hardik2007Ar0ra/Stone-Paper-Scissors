# Stone Paper Scissors Game

A simple Stone Paper Scissors game built using HTML, CSS, and JavaScript as part of my JavaScript learning journey. This project helped me understand DOM manipulation, event handling, and implementing game logic.

# Features
Player vs Computer gameplay
<br>
Random computer choice
<br>
Win / Lose / Draw detection
<br>
Instant result display
<br>
Simple and interactive UI

# Tech Stack
HTML – Structure of the game
<br>
CSS – Styling and layout
<br>
JavaScript – Game logic and interactivity

# What I Learned
DOM manipulation in JavaScript
<br>
Handling click events
<br>
Using random functions in JavaScript
<br>
Writing game logic
<br>
Updating UI dynamically
